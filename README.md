# Growth Family

一个帮助家庭成长的网络应用。

## 项目描述

这是一个专注于家庭成长和亲子关系的web应用，提供了多种功能来促进家庭成员之间的互动和交流。

## 功能特点

- 家庭任务管理
- 成长记录
- 家庭相册
- 亲子互动

## 技术栈

- HTML5
- CSS3
- JavaScript

## 如何使用

1. 克隆仓库
2. 打开 family-os.html 文件
3. 开始使用应用
